# 中文阅读镜像

本目录是 `.agents/` 中人工可读 Markdown 文件的简体中文镜像，只用于查阅。

- `.agents/` 始终是唯一运行时事实源；中英文不一致时以英文文件为准。
- 镜像 `rules/`、`agents/`、各 skill 的 `SKILL.md`，以及 Markdown 参考和模板文件。
- 不镜像脚本、清单、平台配置或其他机器读取文件。
- 不要在 `AGENTS.md`、Cursor、Copilot、Codex 或其他平台配置中引用本目录。
- 不要把本目录加入 `setup-project-agents` 的公共资产清单或同步流程。
- 英文 Markdown 文件发生实质变化时，在同一变更中人工更新对应中文文件。

除本说明外，文件均保持与 `.agents/` 相同的相对路径和文件名，便于逐文件对照。
